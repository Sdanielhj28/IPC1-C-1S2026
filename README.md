# Proyecto 1 - IPC1
Sistema de Inventario para Tienda de Ropa

## Descripción
Este proyecto consiste en el desarrollo de un sistema de gestión de inventario para una tienda de ropa.  
El sistema permite administrar productos, registrar ventas y generar reportes del estado del inventario.

La aplicación fue desarrollada en **Java** utilizando **interfaz gráfica con Swing** en el entorno de desarrollo **NetBeans**.

---

## Funcionalidades del sistema

El sistema cuenta con las siguientes funciones principales:

- Agregar producto
- Buscar producto
- Eliminar producto
- Registrar venta
- Generar reporte de inventario en HTML
- Mostrar datos del estudiante

Cada función es ejecutada mediante botones dentro de la interfaz gráfica.

## Funcionamiento del sistema

1. El usuario ejecuta el programa.
2. Aparece la ventana principal con las opciones del sistema.
3. El usuario puede seleccionar una de las siguientes acciones:
   - Registrar un nuevo producto
   - Buscar un producto existente
   - Eliminar un producto del inventario
   - Registrar una venta
   - Generar un reporte HTML del inventario
4. El sistema actualiza la información del inventario según las acciones realizadas.

---

## Reportes

El sistema genera un archivo **HTML** que contiene la información actual del inventario, incluyendo:

- Código del producto
- Nombre
- Categoría
- Precio
- Cantidad disponible

Este archivo puede abrirse en cualquier navegador web.
